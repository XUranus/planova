# 上传户型图

创建项目后，下一步是上传户型图图片。Planova 使用视觉语言模型（VLM）来解析图片，提取房间几何结构、墙体、门窗及其他结构元素。

## 支持的格式

| 格式 | 扩展名 | 最大大小 |
|--------|-----------|----------|
| JPEG | `.jpg`, `.jpeg` | 50 MB |
| PNG | `.png` | 50 MB |
| PDF | `.pdf` | 50 MB |

:::info
为获得最佳效果，请使用清晰的高分辨率俯视户型图，房间标注清楚、墙体边缘可见。手绘草图也可以使用，但准确度可能较低。
:::

## 上传文件

### 拖放上传

上传页面中央有一个大的 **拖放区域**。

1. 从仪表盘打开一个项目。
2. 从文件管理器中将一个或多个户型图文件拖放到该区域。
3. 悬停时，拖放区域会显示高亮边框效果。

### 浏览按钮

如果你更喜欢手动选择，可以点击拖放区域内的 **浏览文件** 按钮，打开系统原生文件选择对话框。

## 文件列表

上传后，文件会出现在拖放区域下方的可滚动列表中。每条记录显示：

- 上传图片的 **缩略图预览**
- **文件名** 和大小
- 缩略图上的 **解析状态叠加徽标**

## 解析状态

每个上传的文件会依次经历以下状态：

```
pending --> parsing --> completed
                   \-> failed
```

| 状态 | 描述 |
|-------|-------------|
| **pending** | 文件已上传，等待解析。 |
| **parsing** | VLM 正在分析户型图图片。 |
| **completed** | 解析完成。房间几何数据已提取，可进行 3D 生成。 |
| **failed** | 解析遇到错误。请查看错误信息了解详情，可以重试。 |

## 自动解析

如果你已在 [设置](../getting-started/configuration) 中配置了 LLM API 密钥，文件上传后会 **自动** 开始解析，无需手动操作。

如果未配置 API 密钥，文件将保持 **pending** 状态，直到你配置好密钥并手动触发重试。

## 解析失败后重试

如果文件解析失败：

1. 查看文件条目上显示的错误信息。常见原因包括：API 密钥无效、网络超时或图片不可读。
2. 修复对应问题（如在设置中更新 API 密钥）。
3. 点击失败文件条目上的 **重试** 按钮。

## 状态轮询

上传页面会自动每 **2 秒** 轮询后端以刷新解析状态。你会看到状态徽标实时更新，无需手动刷新页面。

:::tip
文件解析期间，你可以导航到其他页面再回来——返回时状态已经更新。
:::

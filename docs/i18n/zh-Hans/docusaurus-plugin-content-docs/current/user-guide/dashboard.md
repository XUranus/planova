# 项目仪表盘

**项目仪表盘**是你打开 Planova 后看到的第一个界面。你可以在同一个地方浏览、创建和管理所有室内设计项目。

## 布局

仪表盘以 **2 列响应式网格** 显示项目。在较宽的屏幕上，两张项目卡片并排显示；在较窄的屏幕上，网格会自动折叠为单列。

每张项目卡片显示：

- **项目名称** 和简短描述
- **状态徽标**，表示当前的生成状态
- 缩略图或占位图片
- 操作按钮（打开、删除）

## 演示项目

Planova 内置了两个演示项目，让你无需上传自己的户型图即可体验完整的工作流程：

| 项目 | 描述 |
|---------|-------------|
| **test_studio** | 一套紧凑的单间公寓户型。适合测试单房间生成、家具摆放和小空间漫游导航。 |
| **test_2br** | 一套两居室公寓，包含多个房间、走廊和门洞。适合测试多房间生成、墙体连接和房间之间的过渡。 |

:::tip
建议先打开一个演示项目，查看完整的 3D 场景效果，再创建自己的项目。这样可以先熟悉查看器的操作方式和检查面板。
:::

## 创建新项目

1. 点击仪表盘网格中的 **+ 新建项目** 卡片。
2. 填写弹窗中的字段：
   - **名称**（必填）-- 项目的简短描述性标题。
   - **描述**（选填）-- 关于空间或设计意图的简要说明。
   - **风格** -- 选择一种 [预设风格](./styles.md)（如现代奢华、北欧等）。后续可以更改。
3. 点击 **创建**。

新项目会立即出现在网格中，状态为 **pending（待处理）**。

## 删除项目

1. 点击项目卡片上的 **删除**（垃圾桶）图标。
2. 弹出 **确认对话框**，要求你确认操作。
3. 点击 **删除**，项目及其所有关联文件（上传的户型图、生成的场景和导出文件）将被永久删除。

:::warning
删除操作不可撤销。在删除项目之前，请确保已导出需要保留的内容。
:::

## 项目状态徽标

每张项目卡片都显示一个带颜色的状态徽标：

| 徽标 | 含义 |
|-------|---------|
| **pending** | 项目已创建，但尚未上传或解析户型图。 |
| **generating** | 3D 场景正在生成中。根据复杂程度，通常需要几秒到几分钟。 |
| **completed** | 生成成功。你可以打开项目并浏览 3D 查看器。 |
| **error** | 生成过程中出现错误。打开项目可查看错误详情，如有需要可重试。 |

状态更新是实时反映的，无需手动刷新仪表盘。

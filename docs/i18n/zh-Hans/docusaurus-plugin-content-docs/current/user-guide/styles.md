# 风格预设

Planova 提供 **6 种室内设计风格预设**，控制生成房间的视觉效果。每个预设定义了一套精心策划的纹理、材质和默认家具美学。

## 可用风格

### 现代奢华

大理石表面、金色金属点缀和流畅的现代家具。想象一下抛光地板、深色纹理石材台面和黄铜配件——高端、国际化的视觉效果。

### 奶油风格

柔和的中性色调，搭配圆润的家具轮廓和柔软质感。暖白色、米色亚麻和弧形沙发营造出温馨、舒适的氛围，适合客厅和卧室。

### 北欧

浅色木地板、白色墙壁和简约家具。灵感源自斯堪的纳维亚设计——简洁的线条、实用的造型和明亮通透的色调，强调简洁与自然光。

### 新中式

深色木格栅屏风、水墨画元素，传统与现代的融合。浓郁的胡桃木色调搭配含蓄的书法风格装饰，呈现精致而有文化底蕴的美学。

### 侘寂

裸露的水泥墙、不完美的手工陶器和大地色系。崇尚不完美与自然老化——风化的木材、粗糙的灰泥和有机形态，安静而内敛。

### 工业风

裸露的砖墙、钢梁和爱迪生灯泡。硬质表面、金属管道置物架和回收木材。带有可见结构元素的城市阁楼风格。

## 选择风格

当你 [创建新项目](./dashboard.md#创建新项目) 时，会出现 **风格选择对话框**，6 种预设以 **3 列网格** 排列。每个单元格显示：

- 预览缩略图
- 风格名称
- 一行描述

点击选择一种风格，然后确认。之后可以在检查器面板的 [场景标签页](./inspector.md#场景标签页) 中更改风格。

## 纹理自定义

选择风格后，你可以在场景标签页的 **纹理自定义面板** 中微调单个纹理。

### 类别

纹理按表面类别组织：

| 类别 | 应用于 |
|----------|------------|
| **Floor** | 房间地板、地面 |
| **Wall** | 内墙和外墙表面 |
| **Ceiling** | 天花板平面 |

### 纹理浏览器

每个类别显示一个 **可水平滚动** 的纹理缩略图条带。左右滚动以浏览该类别的所有可用选项。

- 点击纹理可在 3D 视图中预览。
- 选中的纹理会以边框高亮显示。

### 材质属性

每个纹理都带有控制其在光线下渲染效果的材质属性：

| 属性 | 类型 | 描述 |
|----------|------|-------------|
| `base_color` | Color（十六进制） | 应用于纹理的主色调 |
| `roughness` | Number（0.0 -- 1.0） | 表面的粗糙或光滑程度。0 = 镜面般光滑，1 = 完全哑光 |
| `metalness` | Number（0.0 -- 1.0） | 表面的金属质感。0 = 非金属（塑料、木材），1 = 金属 |

你可以在 [检查器标签页](./inspector.md#可视卡片与-json-编辑器) 的 **Materials** 分区中调整这些值，以获得你想要的效果。

:::tip
大理石地板可尝试 `roughness: 0.15`、`metalness: 0.05` 以获得抛光石材效果。裸露水泥（侘寂风）可尝试 `roughness: 0.85`、`metalness: 0.0`。
:::

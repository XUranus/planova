# 快速开始

本指南将带你完整体验 Planova 的工作流程：上传户型图、审查 AI 解析结果、探索 3D 场景，以及导出最终成果。

## 1. 启动应用

以开发模式启动 Planova：

```bash
pnpm tauri dev
```

应用将打开**项目仪表板**。

## 2. 创建新项目

点击仪表板上的**新建项目**按钮。系统会提示你填写：

- **名称** -- 项目的描述性名称（例如"三居室公寓"）
- **描述** -- 可选备注
- **风格** -- 选择六种室内风格之一：
  - 现代奢华
  - 奶油风
  - 北欧风
  - 新中式
  - 侘寂风
  - 工业风

点击**创建**继续。你将进入项目详情页面。

:::tip 风格可随时更改
你可以随时在 3D 查看器工具栏中更改风格预设。每种风格会应用不同的材质、颜色和家具组合。
:::

## 3. 上传户型图

在项目详情页面，点击上传区域或拖放户型图文件。

**支持的格式：**
- JPG / JPEG
- PNG
- PDF

最大文件大小为 50 MB。

:::info 最佳效果建议
为获得最准确的解析结果，请使用清晰、高分辨率的户型图，房间标注、墙体线条和门窗符号应清晰可见。手绘草图也可以使用，但质量可能较低。
:::

## 4. 等待 AI 解析

上传完成后，Planova 会自动启动解析流水线。你将看到每个阶段的进度指示：

1. **预处理** -- 图像清洗和标准化
2. **VLM 解析** -- AI 视觉模型分析户型图
3. **标准化** -- 将原始检测结果转换为场景图
4. **修复** -- 修复几何问题
5. **验证** -- 对场景进行质量评分
6. **3D 生成** -- 放置家具并应用材质

整个过程通常需要 30-90 秒，具体时间取决于你的 LLM 提供商和户型图的复杂程度。

## 5. 审查解析质量

解析完成后，你将看到 **ReviewGate** 页面。其中显示：

- **质量评分** -- 墙体完整性、房间封闭性和家具覆盖率等指标，以百分比展示
- **调试叠加层** -- 一张渲染图像，将检测到的墙体、房间、门和窗户叠加在原始户型图上
- **场景预览** -- 快速预览生成的 3D 场景

你有两个选择：

| 操作 | 使用场景 |
|------|----------|
| **接受** | 质量评分良好，进入 3D 查看器 |
| **重试** | 评分较低或解析遗漏了房间；重新运行流水线 |

:::warning 质量评分较低
如果墙体完整性或房间封闭性低于 70%，建议重试或上传更清晰的户型图。低分意味着 3D 几何体可能存在间隙或房间形状不正确。
:::

## 6. 探索 3D 场景

接受解析结果后，你将进入 **3D 查看器**。有三种交互模式：

### 环绕模式（默认）
- **左键拖动** 旋转相机围绕场景
- **滚轮** 缩放
- **右键拖动** 平移
- **重置相机** 按钮恢复默认视角

### 漫游模式
- 点击工具栏中的**漫游**进入第一人称模式
- **WASD** 或 **方向键** 移动
- **鼠标** 环顾四周
- 按 **Esc** 退出漫游模式

### 编辑模式
- 点击工具栏中的**编辑**
- **点击任意物体**（家具、墙体等）选中它
- 使用控件对选中物体进行**移动**、**旋转**或**删除**
- 切换**天花板**的显示/隐藏以获得更好的编辑视角

## 7. 更换风格预设

在查看器工具栏中，点击风格选择器切换预设。每个预设会改变：

- 墙面和地面的材质/纹理
- 天花板样式
- 家具模型和颜色
- 灯光氛围

场景会使用新风格重新生成，同时保留房间布局。

## 8. 导出或渲染

在 3D 查看器工具栏中，你有两个导出选项：

### 导出为 GLB

点击 **Export GLB** 将 3D 场景保存为标准 `.glb` 文件。该文件可导入：

- Blender
- Unity
- Unreal Engine
- 任何支持 GLB/GLTF 格式的 3D 工具

### AI 渲染

点击 **AI 渲染**生成当前视角的照片级真实感图像：

1. 可选：编辑渲染提示词，描述你想要的氛围或光照效果
2. 点击**生成**
3. 等待图像生成模型完成渲染
4. 渲染图像将保存到你的项目中

:::info AI 渲染需要配置
AI 渲染使用独立的图像生成提供商。请确保已在设置中配置了**图像提供商**及其兼容模型（如 DALL-E、Stable Diffusion）。
:::
